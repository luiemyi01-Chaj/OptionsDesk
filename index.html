📖 Quick Setup Guide
1. GitHub Pages Hosting
Push index.html to a GitHub repo → Settings → Pages → Deploy from branch (main) → Your app: https://yourusername.github.io/optionsdesk/
2. Google Sheets Database
Create a Google Sheet with tabs: Trades, Watchlist, Settings. Share it (anyone with link can view). Paste the ID above.
3. Google API Key
Google Cloud Console → Create Project → Enable "Google Sheets API" → Credentials → API Key. For write access, add OAuth 2.0 Client ID.
4. Market Data (Required)
Free API key from .com — 800 calls/day, no credit card. Go to Settings → Market Data API → paste key → Save & Test. Options chains use Yahoo Finance as fallback.
